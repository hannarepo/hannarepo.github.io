    Green City is an educational city building game where the player is the mayor of a city and needs to balance the city budget with the pollution levels.
    The game was a first year project in Tampere University of Applied Sciences made for Edusta and the UN Agenda 2030. The goal was to learn basic Unity game development for mobile and agile project management.

    The project is built with Unity 2022 Universal Render Pipeline (URP).

    In this project my main responsibilities were:

  • Implementing building spawning and placement checks
  • Implementing save system
  • Implementing UI functionality
  • Implementing localization
  • Implementing camera functionality with touch controls
