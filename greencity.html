    Green City is an educational mobile city-building game built in Unity 2022 URP for Android. The project was created as a first-year game development project at Tampere University of Applied Sciences for Edusta and the UN Agenda 2030, with a focus on sustainable city planning and learning the basics of mobile game development.

    My main responsibilities included building spawning and placement checks, save system implementation, UI functionality, localization, and camera controls with touch input. Through this project, I gained hands-on experience in building core gameplay and interface systems for a mobile game environment.
