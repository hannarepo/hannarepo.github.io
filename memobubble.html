    • Project

    Memo Bubble is a short 2D platformer in the style of Bubble Bobble. The game has three different worlds and each world has 11 levels.

    The project is built with Unity 2022 Universal Render Pipeline (URP).

    In this project my main responsibilities were:

  • Implementing player mechanics
  • Implementing item collection, shop and power-ups
  • Implementing undefeatable enemy functionality
  • Implementing Hurry up! mechanics

    • Key features

  • Platforming
  • High score
  • Power-ups

    • Player mechanics

    The player has similar functionality as in the old Bubble Bobble versions such as jumping, shooting bubbles and popping them, but Memo Bubble has some added bonuses. Unlike in the classic Bubble Bobble, the player can drop down through platforms and the movement is faster and more responsive.

    Just as in the classic, the player can get power-ups with various effects such as movement speed and bubble size increase, but unlike in the classic I implemented a shop where the player can buy the power-ups with collected points any time. The is also the rarely appearing Umbrella that let's the player skip levels. I also implemented the Hurry up! mechanics familiar from the classic Bubble Bobble. If the player spends too much time in one level, the music speeds up, a Hurry up! prompt is displayed and the enemies get faster. If the player still stays on the level for a while, an undefeatable enemy appers. The enemy will constantly chase the player, ignore all collisions and it cannot be trapped in a bubble and killed. Once the undefeatable enemy reaches the player, one life is lost and the player is respawned.
